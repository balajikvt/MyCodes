Detailed Description

This source file implements a 6 byte addition.

Mathematical operations on numbers larger than those supported by the processor ALU must be carried out by dividing the operands into smaller chunks such as bytes or words. In this particular case, adding two 6 byte numbers can be accomplished by casting them as long long integers (8 byte)
